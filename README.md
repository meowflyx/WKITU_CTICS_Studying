# Учебные работы по программированию (ВТиИС 12-О, ЗКИиТУ)

Данный репозиторий предназначен для хранения моих учебных проектов и домашних заданий.
Здесь собраны решения задач, которые можно запустить для проверки. Для удобства решения разделены на папки в зависимости от автора.

## Как запустить

1.  Убедитесь, что у вас установлен Python версии 3.x.
2.  Склонируйте репозиторий или скачайте файлы.
3.  Запустите интересующий скрипт через терминал:

```bash
python card_check.py
```

Некоторые решения могут быть в отдельных папках. В таком случае в папке будет файл с виртуальным окружением и/или зависимостями. Виртуальную среду стоит активировать перед запуском скрипта.

```bash
/venv/scripts/activate
python card_check.py
```
